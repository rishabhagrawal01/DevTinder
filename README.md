# 🚀 DevTinder

DevTinder is a Tinder-like networking platform built using the MERN stack, where developers can connect with each other, send/receive connection requests, and grow their professional network.

---

## 📌 Features

- 🔐 User Authentication (Signup/Login)
- 👤 Developer Profile Creation & Management
- 🤝 Send Connection Requests
- ✅ Accept / ❌ Reject Requests
- 📬 View Incoming & Sent Requests
- 🌐 Build & Expand Developer Network
- 🔍 Discover Other Developers

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- CSS / SCSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- JWT Authentication
- Git & GitHub
